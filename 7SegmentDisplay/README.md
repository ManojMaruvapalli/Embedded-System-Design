# 7SegmentDisplay
 Using a 7 Segment Display for Embedded Systems Design  
 > Note : We are using STM32F401 ARM board and Arduino UNO R3 board
 
 ## 7Segment_STP  
 It is the code for displaying numbers 0 to 9 in a 7 segment display using a STM32 arm board.

 ## 7Segment_UNO_Type_1/Type2  
 It is the code for displaying numbers 0 to 9 in a 7 segment display using a Arduino UNO R3 arm board.
